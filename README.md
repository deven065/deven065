### [FlatMate](https://github.com/deven065/FlatMate) — Roommate & Expense Management Platform

**The Problem:** Shared living comes with challenges — split bills, shared expenses, task coordination, and communication.

**The Solution:** Built a comprehensive MERN stack application that simplifies roommate management through automated expense splitting, task tracking, and integrated group communication.

**Key Features:**
- 💰 Smart expense splitting with multiple payment methods
- ✅ Task assignment and tracking system
- 💬 Real-time group chat functionality
- 🔐 Secure JWT-based authentication
- 📊 Expense analytics and reporting

**Impact:** Helping 100+ users manage shared living more effectively.

**Tech Stack:** MongoDB, Express.js, React, Node.js, Socket.io, JWT
